Sobre o Produto:

Produto inspirado em uma sistema de hospital fictício, possuindo módulos de cadastro, consulta, overflow de internação, tendo como seu principal objetivo estudos academicos.

Sobre a desenvolvimento do produto:

Utilizando o framework Angular para o desenvolvimento front-end, e utilizando para o back-end api em Java com Spring Boot. O desenvolvimento do produto possui a utilização de design patterns tanto no front-end quanto ao back-end sendo alguns deles, lazy load, repository e MVC.
